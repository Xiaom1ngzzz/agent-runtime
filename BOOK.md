# 本书 — Agent Runtime

## 目标

系统地阐述 **Agent Runtime** 的设计与实现：介于 LLM 与业务之间的运行时层，负责上下文、状态、调度、工具、记忆、可观测性等横切关注点。

读者：

- 正在从零构建 Agent 系统的工程师；
- 已有 Agent 原型、准备走向工程化的团队；
- 想理解 Agent 内部机制的架构师和研究者。

## 写作原则

1. **先讲清问题，再讲解法。** 每一章从一个真实痛点切入。
2. **代码与文字对齐。** `runtime-go/` 与 `runtime-rs/` 中每段参考实现都能在书中找到对应章节；两种语言字段逐一对齐。
3. **决策留痕。** 关键设计通过 ADR 记录 —— 上下文、选项、后果。
4. **面向演进,而非教条。** 给出取舍与替代方案,不追求唯一正解。
5. **可复现。** 图、示例、测试均可在本仓库中重跑;性能基准需单独标注环境与前置条件,不默认等同生产负载。

## 非目标

- 不是某个具体框架的使用手册。
- 不覆盖模型训练与 RLHF。
- 不做 Prompt Engineering 大全。
